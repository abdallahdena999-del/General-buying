# Umma Campus Broker - Jumia-Style Website

This is a Jumia-style e-commerce MVP website for Umma University students.

## Features
- Search bar for products
- Product listing (placeholders)
- Add to Cart functionality
- Checkout options: M-Pesa or Pay after delivery
- Email order notification
- WhatsApp inquiries

## How to Use
1. Upload to GitHub repository.
2. Enable GitHub Pages.
3. Add product images and details in 'images/' folder and update 'js/main.js'.
4. Orders will open your email (mailto) and WhatsApp button can be clicked for inquiries.
